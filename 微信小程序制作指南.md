# 明日方舟通行证统计 - 微信小程序制作指南

## 一、准备工作

### 1. 注册微信小程序账号
1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册" → 选择"小程序"
3. 填写邮箱、密码完成注册
4. 邮箱激活账号
5. 完善小程序信息（名称、头像、介绍等）

### 2. 下载微信开发者工具
1. 访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载对应系统的版本
3. 安装并登录（用注册的微信号扫码）

### 3. 获取 AppID
1. 登录小程序后台 https://mp.weixin.qq.com/
2. 左侧菜单 → 开发 → 开发设置
3. 复制 **AppID(小程序ID)**

## 二、技术方案选择

### 方案对比

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **原生开发** | 性能最好，功能最全 | 学习成本高，开发慢 | ⭐⭐⭐ |
| **Taro** | React语法，多端适配 | 需要学习框架 | ⭐⭐⭐⭐⭐ |
| **uni-app** | Vue语法，多端适配 | 性能略逊于原生 | ⭐⭐⭐⭐ |
| **微信原生 + 云开发** | 官方支持，有云数据库 | 绑定微信平台 | ⭐⭐⭐⭐ |

### 推荐方案：Taro + React

理由：
- 你已有 React 基础
- 代码可以复用大部分逻辑
- 支持编译到微信小程序
- 未来可扩展到其他平台

## 三、使用 Taro 开发（推荐）

### 1. 安装 Taro CLI

```bash
npm install -g @tarojs/cli
```

### 2. 创建项目

```bash
taro init arknights-pass-wx
```

选择：
- 框架：React
- 语言：TypeScript
- 样式：Sass
- 模板：默认模板

### 3. 安装依赖

```bash
cd arknights-pass-wx
npm install
```

### 4. 安装 UI 库

```bash
# 推荐 Taro UI
npm install taro-ui

# 或使用 WeUI
npm install weui-miniprogram
```

### 5. 配置 Supabase

```bash
npm install @supabase/supabase-js
```

创建 `src/lib/supabase.ts`：

```typescript
import { createClient } from '@supabase/supabase-js';

const SUPABASE_URL = 'https://xlegccilynwgqljdvgnk.supabase.co';
const SUPABASE_ANON_KEY = '你的anon_key';

export const supabase = createClient(SUPABASE_URL, SUPABASE_ANON_KEY);
```

### 6. 配置 AppID

修改 `project.config.json`：

```json
{
  "miniprogramRoot": "dist/",
  "projectname": "arknights-pass-wx",
  "description": "明日方舟通行证统计",
  "appid": "你的AppID",
  "setting": {
    "urlCheck": false
  }
}
```

### 7. 开发页面

#### 首页 `src/pages/index/index.tsx`

```tsx
import { View, Text, ScrollView, Input } from '@tarojs/components';
import { useState, useEffect } from 'react';
import { supabase } from '../../lib/supabase';
import './index.scss';

export default function Index() {
  const [boxes, setBoxes] = useState([]);
  const [user, setUser] = useState(null);
  
  useEffect(() => {
    // 加载数据
    loadData();
  }, []);
  
  const loadData = async () => {
    // 从 Supabase 加载或本地 JSON
    const res = await fetch('https://你的域名/data.json');
    const data = await res.json();
    setBoxes(data.standard_boxes);
  };
  
  return (
    <View className="index">
      {/* 标题 */}
      <View className="header">
        <Text className="title">明日方舟通行证统计</Text>
        <Text className="version">v1.0.0</Text>
      </View>
      
      {/* 统计卡片 */}
      <View className="stats">
        <View className="stat-item">
          <Text className="stat-num">0</Text>
          <Text className="stat-label">已拥有</Text>
        </View>
        <View className="stat-item">
          <Text className="stat-num">0%</Text>
          <Text className="stat-label">进度</Text>
        </View>
      </View>
      
      {/* 盒子列表 */}
      <ScrollView className="box-list" scrollY>
        {boxes.map(box => (
          <View key={box.id} className="box-card">
            <View className="box-header">
              <Text className="box-title">通行认证 {box.id}.0</Text>
              <Text className="box-count">0/14</Text>
            </View>
            {/* 展开后显示干员列表 */}
          </View>
        ))}
      </ScrollView>
    </View>
  );
}
```


1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `arknights-pass-wx/dist` 目录
4. 输入 AppID
5. 点击"确定"
